# Data Warehouse Project

This repository contains a **Data Warehouse project** built as part of my journey into **data engineering**.

The goal of this project is to practice and strengthen my **SQL skills** while learning core data warehousing concepts such as:
- Data modeling
- ETL / ELT processes
- Fact and dimension tables
- Querying analytical data

This project was created by following a tutorial by **Data with Baraa**, with a focus on understanding each step rather than simply copying code.

## Technologies Used
- SQL
- Relational Database (PostgreSQL)

## Purpose
This is a **learning-focused project** intended to demonstrate my growing understanding of data engineering fundamentals.
